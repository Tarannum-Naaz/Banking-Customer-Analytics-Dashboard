Banking Customer Analytics Dashboard
Overview

This project presents an interactive Banking Customer Analytics Dashboard developed using Power BI, Python, and SQL. The dashboard provides insights into customer deposits, loans, account balances, and banking relationships to support business decision-making.

Loan Analysis
Deposit Analysis
